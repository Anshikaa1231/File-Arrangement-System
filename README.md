# 📂 File Arrangement System

A **File Arrangement System** is a desktop application designed to automatically organize files into categorized folders based on their file types. The project helps users keep their directories clean and well-structured by reducing manual effort and improving file management efficiency.

## 📖 Overview

Managing large numbers of files can become difficult when documents, images, videos, and other file types are stored in the same location. This File Arrangement System automates the organization process by scanning a selected directory and arranging files into separate folders according to their extensions. The application provides a simple, efficient, and user-friendly solution for maintaining an organized workspace.

---

## ✨ Features

* 📁 Automatically organizes files by type
* 📄 Supports multiple file extensions
* ⚡ Fast and efficient file processing
* 📂 Creates folders automatically if they do not exist
* 🔍 Scans directories for unorganized files
* 🖥️ Simple and easy-to-use interface
* ♻️ Reduces manual file management effort
* 🚀 Improves workspace organization
